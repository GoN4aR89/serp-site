
### 🔐 Настройка администратора:

1. **Способ 1 - Через код:**
   ```bash
   python3 init_db.py
   ```
   Установит админа на email из переменной `ADMIN_EMAIL` в `.env`

2. **Способ 2 - Через Python shell:**
   ```python
   from app import app, db, User
   
   with app.app_context():
       user = User.query.filter_by(email='admin@example.com').first()
       if user:
           user.is_admin = True
           db.session.commit()
   ```

### ❓ Часто задаваемых вопросы:

**Q: Как создать нового администратора?**
A: Зарегистрируйте пользователя, затем в БД установите `is_admin=True` для этого email.

**Q: Что если .env потерян?**
A: Используйте `.env.example` как основу и восстановите данные.

**Q: Понадоб ли пересоздавать БД?**
A: Нет. Скрипт миграции добавит новое поле автоматически.

**Q: Безопасна ли БД на SQLite?**
A: Для небольших проектов - да. Для production используйте PostgreSQL/MySQL.
