ИНСТРУКЦИЯ ПО ЗАПУСКУ САЙТА SERP COMPARATOR НА macOS

1. Установите Python (если ещё не установлен)
   - Откройте Терминал (Finder → Программы → Утилиты → Терминал)
   - Проверьте, есть ли Python:
         python3 --version
     Если показывает версию (например, Python 3.11), переходите к шагу 2.
   - Если Python не установлен, установите через Homebrew или с официального сайта:
         * Homebrew: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
           затем brew install python
         * Или скачайте установщик с python.org/downloads и установите как обычную программу

2. Получите проект
   - Скопируйте папку с проектом (например, serp_site) на свой Mac
   - Если проект в архиве, распакуйте его
   - Запомните путь к папке (например, /Users/имя/Desktop/serp_site)

3. Откройте Терминал и перейдите в папку проекта
   - В Терминале выполните:
         cd /Users/имя/Desktop/serp_site
     (имя замените на своё имя пользователя)

4. (Рекомендуется) Быстрая установка через скрипт:
   - Выполните:
         bash setup.sh
   - Это автоматически создаст виртуальное окружение, установит зависимости,
     создаст .env файл (если его нет) и инициализирует базу данных
   - После выполнения скрипта перейдите к шагу 7 (запуск сервера)

   Или установите вручную (шаги 4-6):

4. (Альтернатива) Создайте виртуальное окружение вручную:
   - Выполните:
         python3 -m venv venv
   - Активируйте окружение:
         source venv/bin/activate
   - В Терминале появится (venv) перед путём

5. Установите необходимые библиотеки
   - Убедитесь, что вы в папке проекта и виртуальное окружение активно
   - Выполните:
         pip install -r requirements.txt
   - Или установите вручную:
         pip install flask pandas openpyxl matplotlib seaborn numpy pytz flask-sqlalchemy python-dotenv
   - Дождитесь окончания установки

6. Настройте конфигурацию (.env файл)
   - Скопируйте пример конфигурации:
         cp .env.example .env
   - Отредактируйте файл .env в текстовом редакторе:
         nano .env
   - Обязательно измените:
     * SECRET_KEY - укажите случайную строку
     * MAIL_USERNAME и MAIL_PASSWORD - для отправки кодов верификации
     * ADMIN_EMAIL - email администратора (доступ к админ-панели)
   - Сохраните файл (Ctrl+O, Enter, Ctrl+X в nano)

7. Инициализируйте базу данных
   - Выполните:
         python3 init_db.py
   - Будет создана база данных и установлен администратор

8. Запустите сервер
   - В Терминале выполните:
         python3 app.py
   - Вы увидите сообщение:
         * Running on http://127.0.0.1:5000
   - Не закрывайте Терминал – пока он открыт, сайт работает

9. Откройте сайт в браузере
   - Запустите Safari, Chrome или другой браузер
   - В адресной строке введите:
         http://127.0.0.1:5000
   - Если не открывается, попробуйте http://localhost:5000

10. Если при открытии сайта появляется ошибка 403 или страница не грузится:
   - Очистите кеш браузера или откройте страницу в режиме инкогнито
   - Перезапустите сервер (Ctrl+C в Терминале, затем снова python3 app.py)

11. Чтобы остановить сервер:
   - В Терминале нажмите Ctrl+C
   - Можно закрыть Терминал

Примечания:
- Все сгенерированные отчёты и загруженные файлы хранятся в папках uploads и user_data внутри проекта.
- При первом запуске создаётся файл базы данных serp_comparator.db.
- Если вы не используете виртуальное окружение, зависимости установятся глобально – это допустимо.
- Для быстрой установки можно использовать скрипт: bash setup.sh (выполнит шаги 4-7 автоматически)
- Версия 3.20.x поддерживает адаптивный дизайн – сайт корректно работает на мобильных устройствах.
- Регистрация требует верификации email – на указанный адрес придёт 6-значный код.
- Версия 3.20.3: оптимизирована структура CSS — все стили вынесены во внешние файлы для улучшения производительности.
