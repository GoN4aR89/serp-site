# ОПИСАНИЕ ФАЙЛОВ ПРОЕКТА SERP Comparator v3.20.3
# =================================================

## 📌 ОСНОВНЫЕ ФАЙЛЫ PYTHON

### app.py
- **Назначение:** Главный файл Flask-приложения. Содержит все маршруты (routes), 
  обработчики запросов, логику авторизации, работу с базой данных, 
  обработку загрузки файлов, генерацию отчетов.
- **Размер:** ~75KB (основной код приложения)
- **Зависимости:** comparator.py, utils.py, constants.py, .env
- **Что делает:**
  * Запускает веб-сервер Flask
  * Обрабатывает HTTP-запросы от пользователей
  * Управляет сессиями и аутентификацией
  * Генерирует Excel-отчеты и графики
  * Работает с БД (SQLite)
- **ЕСЛИ УДАЛИТЬ:** Приложение полностью перестанет работать. Это главный файл.
- **Критичность:** 🔴 КРИТИЧЕСКИЙ

### comparator.py  
- **Назначение:** Модуль для сравнения SERP-данных. Содержит класс SERPComparator
  с методами анализа данных, расчета метрик, визуализации.
- **Размер:** ~62KB
- **Зависимости:** utils.py, constants.py, pandas, matplotlib, seaborn
- **Что делает:**
  * Сравнивает данные из двух Excel-файлов
  * Рассчитывает SERM-индекс, тональность, позиции
  * Создает графики (matplotlib/seaborn)
  * Генерирует Excel-отчеты с несколькими листами
- **ЕСЛИ УДАЛИТЬ:** Функционал сравнения перестанет работать. Приложение запустится,
  но не сможет обрабатывать файлы и генерировать отчеты.
- **Критичность:** 🔴 КРИТИЧЕСКИЙ

### utils.py
- **Назначение:** Утилитарные функции, используемые в app.py и comparator.py.
- **Размер:** ~11KB
- **Зависимости:** Нет (чистые Python-функции)
- **Что делает:**
  * Функции для работы с Excel (форматирование, автоширина)
  * Обработка URL (извлечение домена, очистка)
  * Работа с датами и файлами
  * Утилиты для цветового форматирования ячеек
- **ЕСЛИ УДАЛИТЬ:** comparator.py и app.py не смогут работать (ImportError).
- **Критичность:** 🔴 КРИТИЧЕСКИЙ

### constants.py
- **Назначение:** Файл с константами и настройками проекта.
- **Размер:** ~1.3KB
- **Зависимости:** Нет
- **Что делает:**
  * Содержит словарь SENTIMENT_COLORS для цветов тональности
  * Настройки отступов, размеров, стилей
  * Константы для форматирования Excel
- **ЕСЛИ УДАЛИТЬ:** comparator.py не сможет работать (ImportError).
- **Критичность:** 🔴 КРИТИЧЕСКИЙ

### init_db.py
- **Назначение:** Скрипт инициализации базы данных.
- **Размер:** ~1KB
- **Зависимости:** app.py (импортирует app, db, User)
- **Что делает:**
  * Создает таблицы базы данных при первом запуске
  * Позволяет назначить администратора через переменную окружения ADMIN_EMAIL
  * Вызывается автоматически setup.sh
- **ЕСЛИ УДАЛИТЬ:**
  * База данных не создастся автоматически
  * Пользователи не смогут регистрироваться/входить
  * Решение: можно создать БД вручную через Python shell
- **Критичность:** 🟡 ВАЖНЫЙ (для первоначальной установки)

### migrate_db.py
- **Назначение:** Скрипт миграции базы данных (для обновления существующих БД).
- **Размер:** ~1.7KB
- **Зависимости:** app.py
- **Что делает:**
  * Добавляет новые поля в существующие таблицы БД
  * Например, добавляет поле is_admin для существующих пользователей
  * Используется при обновлении приложения на новую версию
- **ЕСЛИ УДАЛИТЬ:**
  * Обновление БД при переходе на новые версии будет сложнее
  * Придется делать миграцию вручную
- **Критичность:** 🟢 НЕКРИТИЧЕСКИЙ (только для обновлений)


## 📌 ФАЙЛЫ КОНФИГУРАЦИИ

### .env
- **Назначение:** Файл с секретными настройками (не коммитится в git).
- **Размер:** ~650 байт
- **Содержит:**
  * SECRET_KEY (ключ шифрования сессий)
  * MAIL_USERNAME и MAIL_PASSWORD (для SMTP)
  * DATABASE_URL (путь к БД)
  * ADMIN_EMAIL (для автоматического назначения админа)
- **ЕСЛИ УДАЛИТЬ:**
  * Приложение не запустится (ошибка конфигурации)
  * Не будет работать email-верификация
  * Не будет доступа к БД
- **Критичность:** 🔴 КРИТИЧЕСКИЙ
- **Примечание:** Для production обязательно заменить значения!

### .env.example
- **Назначение:** Шаблон .env файла с примерами значений.
- **Размер:** ~600 байт
- **Что делает:**
  * Используется как основа для создания .env
  * Копируется setup.sh если .env не существует
  * Не содержит реальных секретов (безопасно для git)
- **ЕСЛИ УДАЛИТЬ:**
  * Невозможно быстро создать .env через setup.sh
  * Придется создавать .env вручную
- **Критичность:** 🟢 НЕКРИТИЧЕСКИЙ (удобство установки)

### requirements.txt
- **Назначение:** Список Python-зависимостей с версиями.
- **Размер:** ~165 байт
- **Что делает:**
  * Используется pip для установки библиотек
  * Гарантирует совместимые версии Flask, pandas и т.д.
- **ЕСЛИ УДАЛИТЬ:**
  * Сложно установить все зависимости вручную
  * setup.sh не сможет установить пакеты автоматически
- **Критичность:** 🟡 ВАЖНЫЙ

### .gitignore
- **Назначение:** Список файлов, которые git должен игнорировать.
- **Размер:** ~500 байт
- **Что делает:**
  * Исключает .env (секреты), __pycache__, БД, загруженные файлы
  * Предотвращает попадание чувствительных данных в git
- **ЕСЛИ УДАЛИТЬ:**
  * Можно случайно закоммитить .env с паролями в git
  * Репозиторий засорится временными файлами
- **Критичность:** 🟡 ВАЖНЫЙ (для безопасности)

## 📌 ПАПКИ С ФАЙЛАМИ

### templates/
- **Назначение:** HTML-шаблоны Jinja2 для веб-страниц.
- **Содержит:**
  * base.html - базовый шаблон
  * index.html - главная страница (загрузка файлов)
  * compare.html - страница сравнения
  * login.html, register.html - авторизация
  * profile.html - профиль пользователя
  * feedback.html - форма обратной связи
  * admin_feedback.html - админ-панель
  * info.html - информационная страница
  * verify.html - подтверждение email
  * view_comparison.html - просмотр сохраненного сравнения
- **ЕСЛИ УДАЛИТЬ:** Приложение запустится, но все страницы будут с ошибкой 404.
- **Критичность:** 🔴 КРИТИЧЕСКИЙ

### static/css/
- **Назначение:** Внешние CSS-файлы (вынесены из HTML в v3.20.3).
- **Содержит:**
  * base.css - базовые стили для всех страниц
  * auth.css - стили авторизации
  * compare.css - стили сравнения
  * profile.css, feedback.css, admin_feedback.css, info.css, verify.css, view_comparison.css
- **ЕСЛИ УДАЛИТЬ:** Страницы будут без стилей (некрасивые, но работоспособные).
- **Критичность:** 🟡 ВАЖНЫЙ (функционал сохранится, UI пострадает)

### static/img/
- **Назначение:** Изображения (логотипы, иконки, загруженные графики).
- **ЕСЛИ УДАЛИТЬ:** 
  * Не критично - графики генерируются заново
  * Логотип пропадет с главной страницы
- **Критичность:** 🟢 НЕКРИТИЧЕСКИЙ

### static/js/
- **Назначение:** JavaScript файлы.
- **ЕСЛИ УДАЛИТЬ:**
  * Интерактивность снизится (всплывающие подсказки, динамика)
  * Базовый функционал сохранится

### uploads/
- **Назначение:** Временная папка для загружаемых файлов Excel.
- **Содержит:** Временные файлы пользователей
- **ЕСЛИ УДАЛИТЬ:** 
  * Создастся автоматически при следующей загрузке
  * Текущие несохраненные данные потеряются
- **Критичность:** 🟢 НЕКРИТИЧЕСКИЙ (временные файлы)

### user_data/
- **Назначение:** Постоянное хранилище для сохраненных отчетов.
- **Содержит:** Excel-отчеты, PDF, сгенерированные графики
- **ЕСЛИ УДАЛИТЬ:**
  * Все сохраненные отчеты пользователей будут потеряны
  * Ссылки на старые отчеты перестанут работать
- **Критичность:** 🟡 ВАЖНЫЙ (пользовательские данные)

### instance/
- **Назначение:** Стандартная папка Flask для хранения БД.
- **Содержит:** Файл БД SQLite (serp_comparator.db)
- **ЕСЛИ УДАЛИТЬ:**
  * База данных будет создана заново
  * Все пользователи и история сравнений будут потеряны!
- **Критичность:** 🔴 КРИТИЧЕСКИЙ (если удалить с БД внутри)

### __pycache__/
- **Назначение:** Кэш скомпилированного Python-кода.
- **Содержит:** .pyc файлы
- **ЕСЛИ УДАЛИТЬ:**
  * Создастся автоматически при следующем запуске
  * Первый запуск будет чуть медленнее
- **Критичность:** 🟢 НЕКРИТИЧЕСКИЙ (можно смело удалять)
